# DearBaby

A warm, minimalist app for expectant parents and families to document pregnancy,
birth, and childhood memories — mobile (iOS + Android), backend, and admin.

This repository is the **project foundation**: architecture, a complete database
schema, the API map, a runnable backend auth core, and the mobile design system.
It is a starting point to build on phase by phase — not a finished product.

## Map
- `docs/ARCHITECTURE.md` — stack choices, system design, offline sync, payments, security
- `docs/API.md` — full endpoint reference across all 12 feature areas
- `docs/ROADMAP.md` — phased plan (ship the memory spine first)
- `backend/` — FastAPI app (auth works today) + `schema.sql` (all features)
- `design/` — mobile design system + high-fidelity screen mockups

## Feature areas (from the brief)
Auth · Pregnancy journey · Baby memory book · Photo/video gallery · Digital
scrapbook · Printable books · Community · Recipes · Notifications · Subscriptions
· Settings · Admin dashboard.

## Where to start
Read `docs/ROADMAP.md`, run the backend (`backend/README.md`), then build Phase 1.
